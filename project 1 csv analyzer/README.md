<h2>Project 1</h2> CSV Data Analyzer | Difficulty: 1/10 |
Duration: 2-3 days | Tech: Pandas, Python <br>
What You Build: A Python script that reads a CSV file, counts rows, calculates basic statistics (mean, median, sum) for numeric columns, and displays
them beautifully.<br>
Exact Steps:<br> 1. Load a CSV file using Pandas (use sample Kaggle dataset
like sales data) <br>2. Display total rows and columns<br> 3. Calculate mean, median,
mode for each numeric column<br> 4. Calculate sum and count<br> 5. Display results
in a nice table format<br> 6. Save results to a new CSV file<br>
Learning Goals: - File I/O operations - CSV reading with Pandas - Basic
data manipulation - Working with DataFrames<br>
Real-World Use: Business analytics, data reporting, sales data analysis dash-
boards
